## Learn ADO.NET

This repository is created for **learning and practicing ADO.NET** using **C#**.

The repository contains multiple folders.  
Each folder represents a **separate Visual Studio project**, and each project demonstrates **a specific ADO.NET concept**.

All projects are built using **Visual Studio** and are intended for **learning and hands-on practice**, not for production use.

### Database Backup

The file **ContactsDB.bak** is a **SQL Server database backup** used for practice in this repository.  
You can restore this database locally to run and test the projects.
